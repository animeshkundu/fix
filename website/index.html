Install

Downloads the latest release and installs to ~/.local/bin

iwr -useb https://animeshkundu.github.io/fix/install.ps1 | iex

Run in PowerShell. Installs to %LOCALAPPDATA%\fix and adds to PATH.

cargo install --git https://github.com/animeshkundu/fix fix-cli

Requires Rust toolchain. Add --features metal on macOS for GPU acceleration.

Download the binary for your platform from GitHub Releases:

  • fix-aarch64-apple-darwin.tar.gz macOS Apple Silicon
  • fix-x86_64-apple-darwin.tar.gz macOS Intel
  • fix-x86_64-unknown-linux-gnu.tar.gz Linux x64
  • fix-x86_64-pc-windows-msvc.zip Windows x64

How it works

1

Download

On first run, fix downloads a small language model (~400MB) fine-tuned for shell command correction.

2

Run locally

The model runs entirely on your machine using Metal (macOS) or CUDA (Linux/Windows) for fast inference.

3

Stay private

Your commands never leave your computer. No telemetry, no API calls, no data collection.

Usage

$ fix --help
Fix shell command typos using a local LLM

Usage: fix [OPTIONS] [COMMAND]...

Arguments:
  [COMMAND]...  The command to correct

Options:
  -e, --error <ERROR>      Error message from failed command
  -s, --shell <SHELL>      Override detected shell
      --list-models        List available models
      --use-model <NAME>   Download and set default model
      --show-config        Show current configuration
  -v, --verbose            Show model loading logs
  -h, --help               Print help
  -V, --version            Print version
