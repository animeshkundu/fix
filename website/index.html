cmd-correct

Fix shell typos instantly.

$ cmd-correct "gti status"
git status

Install

curl -fsSL https://animeshkundu.github.io/oops-llm-cli/install.sh | sh
cargo install cmd-correct

Requires Rust toolchain. On macOS, add --features metal for GPU acceleration.

Download the latest binary from GitHub Releases.

  • cmd-correct-aarch64-apple-darwin.tar.gz - macOS Apple Silicon
  • cmd-correct-x86_64-apple-darwin.tar.gz - macOS Intel
  • cmd-correct-x86_64-unknown-linux-gnu.tar.gz - Linux x64
  • cmd-correct-x86_64-pc-windows-msvc.zip - Windows x64

How it works

  • Downloads a small language model on first run (~400MB)
  • Runs entirely locally — no API keys, no internet after setup
  • Sub-100ms inference on Apple Silicon with Metal GPU
  • Trained on 150K shell command corrections

Usage

$ cmd-correct --help
Fix shell command typos using a local LLM

Usage: cmd-correct [OPTIONS] [COMMAND]...

Arguments:
  [COMMAND]...  The command to correct

Options:
  -e, --error <ERROR>      Error message from failed command
  -s, --shell <SHELL>      Override detected shell
      --list-models        List available models
      --use-model <NAME>   Download and set default model
      --show-config        Show current configuration
  -h, --help               Print help
  -V, --version            Print version
